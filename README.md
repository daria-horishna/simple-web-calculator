\# Simple Web Calculator



\*\*Базовий веб-калькулятор для виконання простих арифметичних операцій.\*\*



Це MVP проєкту, який містить два поля для введення чисел та можливість вибору математичної дії (+, -, \*, /).



\## Стек технологій

\* HTML5

\* CSS3

\* Vanilla JavaScript




\## Інструкція з запуску

1\. Склонуйте репозиторій на свій комп'ютер.

2\. Відкрийте файл `index.html` у браузері.
